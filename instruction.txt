Инструкция по поднятию бэкенда для проекта BudgetBuddy(домашняя бухгалтерия):
- Установить pg admin (postgresql на пк) - https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
- Создать БД через pg admin
- Обновить данные в env - DATABASE_URL=postgresql://postgres:123456@localhost:5432/auth-mk?schema=public

Name: auth-mk - название БД
Host name / address: localhost
Port: 5432 - дефолтный порт
Username: postgres - юзернейм postgresql
Password: 123456 - пароль postgresql

Запуск:
	npm install
	npm prisma db push

Google
Для получения ключей Google OAuth выполнить следующие шаги:

Перейти на Google Cloud Console(Google Develop Console - https://console.cloud.google.com/).
Создайть новый проект или выбрать существующий.
Перейти в меню APIs & Services > Credentials (https://console.cloud.google.com/apis/dashboard).
Нажать Create Credentials и выберите OAuth Client ID.
Настроить экран согласия OAuth (external), если это еще не сделано.

Заполнить необходимые поля (auth domains пропускаем, как и вкладку scopes и optional info).
Добавить google аккаунт, который будет тестировать в поле "Test users"
Вернуться на страницу https://console.cloud.google.com/apis/credentials и нажать кнопку Create credentials -> OAuth client ID
Выбрать тип веб-приложение
И далее заполнить поля (или свой домен на production
origins = http://localhost:4200
redirects = http://localhost:4200/auth/google/callback)
После этого нажать кнопку создать и получить Client ID и Client Secret.

GitHub
Для получения ключей GitHub OAuth выполнить следующие шаги:

Перейти на GitHub Developer Settings (https://github.com/settings/developers).
В разделе OAuth Apps нажать New OAuth App.
Заполнить необходимые поля, такие как Application name, Homepage URL, и Authorization callback URL.
Нажать Register application и получить Client ID и Client Secret.
